## 🧰 Troubleshooting

### The app does not open

Try these steps:

- Right-click the file and choose Run as administrator
- Check that the file finished downloading
- Make sure your antivirus did not block it
- Re-download the file from the GitHub page

### Windows shows a security prompt

This can happen with new apps on Windows.

- Click More info if needed
- Then choose Run anyway if you trust the source

### The app closes right away

Try this:

- Move the file to a simple folder like Desktop
- Avoid paths with special characters
- Run the app as admin
- Check if the download is a full release file

### The loader does not apply changes

Try these steps:

- Restart the app
- Confirm you selected a preset
- Check that you have permission to make system changes
- Restart Windows after loading
